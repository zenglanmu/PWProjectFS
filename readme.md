 5. 记事本编辑文本文件时，保存会提示找不到文件
 6. 记事本打开文件时，会走只读模式。而pw端没有检出，如果再进一步保存则出错（反正保存也会提示找不到文件）
 7. wps保存文件，因为会创建一堆临时文件，导致速度明显偏慢
